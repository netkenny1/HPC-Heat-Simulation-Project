## Files Included

- `visualize_heat.py` - Standalone VTK visualization script
- `visualize_heat_colab.py` - Google Colab version using Matplotlib
- `Part3_Report.md` - Complete report
- `heat_output.vtk` - VTK output file (2.2MB, 500×500 grid)

## How to Use

### Standalone Visualization

```bash
# Install VTK
pip install vtk

# Run visualization
python visualize_heat.py

# Or specify file and save image
python visualize_heat.py heat_output.vtk output.png
```

### Google Colab

```python
!pip install vtk matplotlib numpy
from visualize_heat_colab import visualize_heat_colab
visualize_heat_colab('heat_output.vtk', 'heat_visualization.png')
```

## VTK File Format

The `heat_output.vtk` file contains:
- Grid dimensions: 500×500
- Temperature data: 250,000 points
- Format: Legacy VTK ASCII format
- Generated by: Parallel version (MPI + OpenMP)

## Visualization Features

- Color-mapped temperature distribution
- Color bar showing temperature scale
- Interactive viewing (standalone version)
- Statistics display (Colab version)
